PBS Schedule API
================

== Introduction ==
A Drupal 7 module that implements the PBS Schedule API.

== Installation ==
Install as any other module.

== Configuration ==
Grant permission "Administer PBS Schedule API" to user role who should edit
the API settings.

Next, go to admin/config/pbs_schedule_api to set the API key and Call sign. If you
do not have a key see: https://projects.pbs.org/confluence/display/tvsapi/TV+Schedules+Version+2
